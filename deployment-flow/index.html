Deployment Flow

This page outlines the steps involved in the deployment process.

  1. Write code
  2. Commit changes
  3. Push code to GitHub
  4. Jenkins triggers build
  5. Build Docker images
  6. Push to DockerHub
  7. Apply Kubernetes deployment
  8. LoadBalancer exposes your app to the world
  9. Scalable, monitored, cloud-hosted MERN app
